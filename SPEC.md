# zndmonya — 非公開ファイルやり取りサイト

## 概要

zndmonya.com は**非公開**のファイルやり取りホームページです。
自宅PCからアップロードし、社内PCからダウンロードする用途を想定しています。

## 運用フロー

```
【自宅PC】ログイン → アップロード
【社内PC】ログイン → ダウンロード
```

## セキュリティ

| 対策 | 内容 |
|------|------|
| 認証 | パスワード必須（全ページは `/` がメイン、ログイン後のみ） |
| 非公開 | `noindex` + robots.txt + 検索エンジン除外 |
| 通信 | HTTPS |
| 推奨 | ZIP パスワード暗号化 |

## 環境変数

| 変数 | 必須 | 説明 |
|------|------|------|
| `VAULT_PASSWORD` | ○ | ログインパスワード |
| `SESSION_SECRET` | ○ | セッション署名用 |
| `BLOB_READ_WRITE_TOKEN` | 本番 | Vercel Blob（未設定時はローカル保存） |

## ローカル開発

```bash
npm run dev:clean
```

ファイルは `packages/` に保存されます（Blob トークン未設定時）。

## Vercel デプロイ

1. Root Directory は **空（リポジトリ直下）** — 以前 `vault` にしていた場合はクリア
2. 環境変数を設定
3. Vercel Blob を接続（本番）
4. ドメイン `zndmonya.com` を割り当て
